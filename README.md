# PreCheck Security - Prototipo Fase 1

Este proyecto es un sistema de validación de seguridad modular diseñado para el reconocimiento de archivos maliciosos mediante el uso de hashes SHA-256. El sistema permite escanear archivos y compararlos con una base de datos de amenazas conocidas.

## 🚀 Características
* **Escaneo de Malware**: Simulación de escaneo basada en firmas digitales únicas.
* **Interfaz Gráfica (GUI)**: Entorno visual intuitivo para el usuario.
* **Arquitectura Modular**: Código organizado por capas (Lógica, Interfaz y Pruebas).
* **Seguridad de Datos**: Implementación de algoritmos SHA-256 para integridad de archivos.

## 🛠️ Tecnologías utilizadas
* **Python 3.x**: Lenguaje base del proyecto.
* **Hashlib**: Biblioteca para la generación de hashes SHA-256.
* **Git/GitHub**: Control de versiones y hosting de código.

## 📋 Requisitos Previos
Antes de ejecutar el programa, asegúrate de tener instalado:
* [Python 3.10+](https://www.python.org/downloads/)
* Git (opcional para clonar)
